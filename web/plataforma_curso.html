O que é CSS?

CSS (Cascading Style Sheets) dá estilo ao HTML: cores, fontes, tamanhos, espaçamentos e até animações.

Exemplo:

 body { background-color: #f0f0f0; font-family: Arial, sans-serif; }

h1 {
color: blue;
text-align: center;
}

p {
color: #333;
line-height: 1.5;
}

Como eles funcionam juntos?

O HTML cria a estrutura, o CSS deixa bonito e organizado.

Exemplo completo:


<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <meta charset="UTF-8">
    <title>Exemplo</title>
    <style>
      body {
        background: #fafafa;
        font-family: Verdana;
      }

      h1 {
        color: darkred;
      }

      p {
        color: #555;
      }
    </style>
  </head>
  <body>
    <h1>Olá, mundo!</h1>
    <p>Este é meu primeiro site com HTML e CSS.</p>
  </body>
</html>
